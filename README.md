# Elegram｜LINE予約フォーム（v2）

エステサロン「Elegram」のLINE予約フォーム（v2）。

## 構成
```
index.html        予約フォーム（単体で動作）
assets/
  img/            ロゴ・ヒーロー写真・装飾・テクスチャ
```

## 使い方
- `index.html` をブラウザで開けば表示されます。
- GitHub Pages で公開する場合は、この一式をリポジトリ直下に置けばそのまま動作します。

## 外部連携（稼働中）
- LINE LIFF（liffId: `2006377601-BWadbvyJ`）
- 予約データ送信先：Google Apps Script（`script.google.com/.../exec`）
- フォント：Google Fonts ／ アイコン：Font Awesome

> ※LIFF ID・送信先URLはこのサロン用に直書きされています。別アカウントで使う場合は差し替えが必要です。
